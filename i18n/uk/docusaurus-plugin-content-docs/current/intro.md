# Svaroh Smart Home

Внутрішня база знань екосистеми Svaroh.

## Що таке Svaroh?

Self-hosted платформа розумного дому з опціональною хмарною інтеграцією. Станції працюють на Raspberry Pi, пристрої — на ESP32, мобільний застосунок забезпечує віддалене керування.
