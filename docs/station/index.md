# 🏠 Station

The Raspberry Pi side of the system. Backend (Fastify), Frontend (React SPA), Postgres, MQTT broker, and the native `station-agent` that manages them.

Three sections:

- [Common](/station/common/) — what is a station, monorepo layout, hardware, station-agent
- [Backend](/station/backend/) — modules, DDD domain, integrations (BLE, MQTT, Cloud)
- [Frontend](/station/frontend/) — FSD architecture, Redux, UI kit + dashboard
